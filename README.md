### Installation

- Just, Clone this repository -
```
git clone https://github.com/TechieGamer1/TechieAdvPhishing
```

- Change to cloned directory and run `TechieAdvPhishing.sh` -
```
cd TechieAdvPhishing
bash TechieAdvPhishing.sh
```
